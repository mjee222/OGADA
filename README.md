# OGADA

국가별 출입국 신고서에 대한 기입방법을 알려주는 앱
- 여권, 비행기 표 등을 찍어 정보 저장
- 원하는 나라 선택 후 가이드 제공

@ 본 프로젝트는 군산대학교 소프트웨어융합공학과 IoT기반 Android프로그래밍 과목의 텀 프로젝트입니다.


# 개발환경

* os : Window 10
* develop tool : Android Studio3.5
* tess-two : 9.0.0
* OpenCV : 3.1.0
